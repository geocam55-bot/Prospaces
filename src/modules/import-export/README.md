# Import/Export Module

This directory contains the core files for the ProSpaces CRM Import/Export system.

## Structure
- `api/` — Backend API handlers (upload, mapping, validation, import, export, jobs)
- `components/` — React UI components (ImportScreen, ExportScreen, MappingGrid, Dashboard, etc.)
- `utils/` — Utilities (Excel parsing, mapping, validation, chunking, logging)
- `types.ts` — TypeScript interfaces and types for schemas, jobs, mappings, etc.

## Getting Started
- See the design blueprint for architecture and flow.
- Start with the provided skeletons and expand each part as needed.
